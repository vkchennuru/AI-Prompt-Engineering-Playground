Reusable Prompt Patterns

Patterns are battle-tested prompt structures. Instead of reinventing the wheel, use proven frameworks.

Expert Persona Pattern‚ñº
You are a [EXPERT_TYPE] with [X] years of experience in [FIELD]. Your expertise includes: [SKILL_1], [SKILL_2], [SKILL_3]. Task: [SPECIFIC_TASK] Requirements: - [REQUIREMENT_1] - [REQUIREMENT_2] Respond using the vocabulary and reasoning style of a [EXPERT_TYPE].
Recipe Pattern‚ñº
To accomplish [GOAL], follow this recipe: Ingredients: - [COMPONENT_1] - [COMPONENT_2] - [COMPONENT_3] Steps: 1. First, [ACTION_1] 2. Then, [ACTION_2] 3. Finally, [ACTION_3] Expected output: [RESULT_DESCRIPTION]
Reflection Pattern‚ñº
Complete this task: [TASK] After completing, critically review your own output: - What assumptions did you make? - What could be improved? - Rate your response quality 1-10 - Provide an improved version Show the initial response, then the reflection, then the improved version.

Prompt Chaining

Prompt chaining breaks complex tasks into a sequence of smaller prompts where each output feeds into the next. This is how production AI workflows are built.

Example Chain: Blog Post Creation

Prompt 1 ‚Üí Outline
Generate a detailed outline for a blog post about [TOPIC]
Prompt 2 ‚Üí Draft
Using this outline: [OUTPUT_1], write the full blog post draft
Prompt 3 ‚Üí Edit
Edit this draft for clarity and engagement: [OUTPUT_2]
Prompt 4 ‚Üí SEO
Generate meta description and title tags for: [OUTPUT_3]

Meta-Prompting

Meta-prompting means using the AI to write, improve, or generate prompts for you. It's a force-multiplier technique.

Meta-Prompt Templates
