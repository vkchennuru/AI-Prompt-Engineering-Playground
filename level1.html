What is a Prompt?

A prompt is any text you send to an LLM (Large Language Model) to get a response. Think of it as giving instructions to a very smart, very literal assistant.


Large Language Models predict the most statistically likely next tokens given your input. This means:

  • Vague prompts ‚Üí vague, generic outputs
  • Specific prompts ‚Üí focused, useful outputs
  • Context-rich prompts ‚Üí contextually accurate outputs

The Golden Rules

Be Specific
Tell the model exactly what you want ‚Äî format, length, tone, audience.
Give Context
Who are you? What's the use case? What constraints exist?
Set Expectations
Specify the output format (list, JSON, paragraph, table).
Iterate
Your first prompt is rarely your best. Refine and retry.

How LLMs Actually Work

LLMs don't "understand" like humans. They complete patterns. Your prompt is the beginning of a document ‚Äî the model completes it. Framing your prompt like a document beginning you want completed is a powerful mental model.


Example: Instead of asking "What is Python?" try starting with: Python is a programming language best suited for... ‚Äî the model will complete this with rich, accurate information.

The Anatomy of a Great Prompt

Every strong prompt typically contains some combination of these elements:

1. Role
You are a [expert type] with [experience]...
2. Task
Write / Analyze / Summarize / Generate...
3. Context
For [audience], in the context of [situation]...
4. Format
As a [list/table/JSON/paragraph]...
5. Length
In [X] words / [N] bullet points...
6. Tone
Using [formal/casual/expert/friendly] tone...

Prompt Template: The Universal Starter

You are a [ROLE] with expertise in [DOMAIN]. Task: [SPECIFIC TASK] Context: [BACKGROUND INFORMATION] Requirements: - Output format: [FORMAT] - Length: [LENGTH] - Audience: [TARGET AUDIENCE] - Tone: [TONE/STYLE] [OPTIONAL: Examples of what you want]
